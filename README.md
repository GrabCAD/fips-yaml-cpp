# fips-yaml-cpp
FIPS-ifed version of yaml-cpp.
* [yaml-cpp](https://github.com/jbeder/yaml-cpp)
* [fips](https://github.com/flooh/fips)


To include this library, edit your fips.yml to import:

```
imports:
    fips-yaml-cpp:
        git: https://github.com/MODit3D/fips-yaml-cpp
```
